## SmartHomeApp

스마트홈앱은 사용자에게 스마트한 방법을 제시해 삶의 질을 높일 수 있는 앱입니다.\
이용자는 실시간으로 방의 온습도 정보와 전등을 on/off 할 수있고, 거실에 있는 어항에 시간마다 원격으로 먹이를 줄 수 있는 편리한 기능을 제공합니다.\
